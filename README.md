## CAN-based Communication Application using UDS Protocol

### Overview
This project demonstrates a simple CAN-based communication application utilizing the UDS protocol.

### Hardware Setup
- **Arduino UNO (Sender)**
- **2 MCP2515 Transceivers**
- **ESP32 (Receiver)**

### Libraries Used
- mcp_can.h
- mcp2515.h
- SPI.h
